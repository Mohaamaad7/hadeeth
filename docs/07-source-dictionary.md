## إضافة مصدر جديد

### الطريقة 1: من لوحة التحكم (مباشر)
1. اذهب إلى **لوحة التحكم > المصادر > إضافة مصدر**
2. أدخل: الاسم الكامل، الرمز، النوع
3. الـ `HadithParser` يتعرف عليه **فوراً** (يحمّل من DB)

### الطريقة 2: من الـ Seeder (للتركيب الأولي)
```php
// database/seeders/SourceSeeder.php
$sources = [
    // ... المصادر الموجودة
    ['name' => 'مسند الطيالسي', 'code' => 'الطيالسي', 'type' => 'مسند'],
];
```
ثم:
```bash
php artisan db:seed --class=SourceSeeder
```

> **ملاحظة:** الـ Seeder يستخدم `firstOrCreate` فلن يُكرر المصادر الموجودة.

### بعد إضافة مصدر جديد:
```bash
# إعادة ربط المصادر بالأحاديث الموجودة
php artisan hadiths:reparse --dry-run   # معاينة أولاً
php artisan hadiths:reparse             # تطبيق
```

## نموذج البيانات

### جدول `sources`
| العمود | النوع | الوصف |
|---|---|---|
| `id` | bigint PK | المعرف |
| `name` | varchar | الاسم الكامل |
| `code` | varchar UNIQUE | الرمز المختصر |
| `type` | varchar | نوع المصدر |
| `created_at` | timestamp | تاريخ الإنشاء |
| `updated_at` | timestamp | تاريخ التحديث |

### جدول `hadith_source` (Pivot)
| العمود | النوع | الوصف |
|---|---|---|
| `hadith_id` | bigint FK | معرف الحديث |
| `source_id` | bigint FK | معرف المصدر |
